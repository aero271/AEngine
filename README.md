<h1>AEngine</h1>
<p>AEngine is a lightweight game engine for building small, fast games using an easy to understand framework. AEngine is built off Raylib, a purely C OpenGL abstraction, though its functionality is expanded upon significantly. UI is handled with
litehtml, a lightweight HTML/CSS renderer, allowing the use of simple web technolgies for UI creation</p>
<h2>NOTE</h2>
<p>AEngine is not complete, and is under active development.</p>
